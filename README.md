# 🤖 AI Agents Skills — Gemini CLI Edition

A collection of **Agent Skills** built for [Google Gemini CLI](https://github.com/google-gemini/gemini-cli), designed to supercharge marketing automation, content generation, SEO, and MSME growth workflows.

> Built by [Sheevum Goel](https://github.com/sheevu) — Founder, NAVA-NETRA NEURAL Sudarshan AI Labs Pvt. Ltd., Lucknow, India.

---

## 📦 Skills Index

| Skill | Description |
|---|---|
| [`gemini-cli-setup`](skills/gemini-cli-setup/SKILL.md) | Install, authenticate, and configure Gemini CLI |
| [`gemini-cli-copywriting`](skills/gemini-cli-copywriting/SKILL.md) | Generate marketing copy, ads, and taglines via CLI |
| [`gemini-cli-seo`](skills/gemini-cli-seo/SKILL.md) | Bulk SEO tasks: meta tags, alt text, schema via CLI |
| [`gemini-cli-cold-email`](skills/gemini-cli-cold-email/SKILL.md) | Draft and automate cold email sequences via CLI |
| [`gemini-cli-social`](skills/gemini-cli-social/SKILL.md) | Generate social media captions and posts via CLI |
| [`gemini-cli-content-strategy`](skills/gemini-cli-content-strategy/SKILL.md) | Build content calendars and strategy via CLI |
| [`gemini-cli-lead-magnets`](skills/gemini-cli-lead-magnets/SKILL.md) | Create lead magnets (ebooks, checklists) via CLI |
| [`gemini-cli-msme-growth`](skills/gemini-cli-msme-growth/SKILL.md) | India-specific MSME marketing automation via CLI |
| [`gemini-cli-analytics`](skills/gemini-cli-analytics/SKILL.md) | Generate reports and insight summaries via CLI |
| [`gemini-cli-ad-creative`](skills/gemini-cli-ad-creative/SKILL.md) | Generate ad headlines and creative briefs via CLI |

## 🚀 Quick Start

```bash
npm install -g @google/generative-ai-cli
export GEMINI_API_KEY=your_key_here
gemini prompt "Write a marketing email for an AI SaaS product"
```
